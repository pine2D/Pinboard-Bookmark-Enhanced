# Pinboard Bookmark Enhanced

[English](README.md) | [简体中文](README.zh-CN.md) | [繁體中文](README.zh-TW.md) | [繁體中文（香港）](README.zh-HK.md) | [Deutsch](README.de.md) | [Français](README.fr.md) | [日本語](README.ja.md) | [Polski](README.pl.md) | **Русский**

Расширение Chrome для [Pinboard](https://pinboard.in): AI-теги и краткое содержание, встроенный режим чтения с переводом и выделениями, а также 13 тем для самого сайта.

> **Примечание:** Требуется аккаунт [Pinboard](https://pinboard.in) — Pinboard (pinboard.in) — это независимый, **платный** сервис закладок. Это расширение является сторонним клиентом, который подключается к вашему существующему аккаунту Pinboard с помощью вашего собственного API-токена Pinboard. Оно не связано с Pinboard, не спонсируется и не одобряется им. Чтобы пользоваться этим расширением, у вас уже должен быть (или нужно оформить) платный аккаунт Pinboard.in.

## Возможности

### Сохранение
- **Один клик — всё заполнено** — заголовок, описание и выделенный текст подставляются сами, а трекинг-параметры удаляются из URL
- **Сохранение горячей клавишей** — без открытия окна; можно сохранить и все открытые вкладки разом
- **Работает офлайн** — записи попадают в локальную очередь и отправляются повторно после восстановления связи

![Сохранение в один клик, AI-теги и краткое содержание](docs/cws-assets/originals/screenshot-1-save.png)

### Теги
- **AI-теги и краткое содержание** — читается текст статьи без рекламы, меню и боковых панелей; свой API-ключ, 13 провайдеров или любой OpenAI-совместимый эндпоинт
- **Автодополнение** — из ваших тегов, подсказок Pinboard и пресетов в один клик
- **Порядок в тегах** — найдите дубликаты и редко используемые теги и объедините их партиями

### Чтение
- **Любая страница — в удобный вид для чтения** — Markdown-просмотр с оглавлением, поиском и предпросмотром сносок
- **Выделения пяти цветов с заметками** — и выделения, и заметки переживают перерисовку, перевод и даже изменения на странице
- **Переведите страницу или задайте ей вопрос** — полный перевод с двуязычным видом; ответы ссылаются на источник, и клик ведёт прямо к нему
- **Отправить или скачать** — в [Obsidian](https://obsidian.md), GitHub Gist или на любой вебхук; либо скачать `.md`, `.html`, `.epub` для читалки

![Режим чтения с двуязычным переводом и выделениями](docs/cws-assets/originals/screenshot-2-reader.png)

### Персонализация
- **13 тем для pinboard.in** (Dracula · Nord · Catppuccin · Solarized · …) плюс свой CSS
- **Автоархив в [Wayback Machine](https://web.archive.org)** — по желанию при каждом сохранении; страницы остаются доступны, даже когда оригинальная ссылка умирает
- **Резервная копия настроек** — экспорт в файл или автоматический бэкап на свой WebDAV-сервер
- **9 языков** · настраиваемые горячие клавиши · локальное хранилище в приоритете · никакого трекинга

## Установка

**[→ Установить из Chrome Web Store](https://chromewebstore.google.com/detail/pinboard-bookmark-enhance/pnjndmjhljjbdlbejeenkepdalokfooh)** — рекомендуется

Или загрузите распакованное расширение из ZIP с релизом:
1. Скачайте последний [ZIP с релизом](https://github.com/pine2D/Pinboard-Bookmark-Enhanced/releases/latest)
2. Распакуйте
3. `chrome://extensions/` → включите **Режим разработчика** → **Загрузить распакованное расширение** → выберите распакованную папку

После установки: нажмите на иконку в панели инструментов → вставьте свой [API-токен Pinboard](https://pinboard.in/settings/password) → сохраните

## Конфиденциальность

Никакого трекинга, аналитики или телеметрии. Для новых пользователей настройки и учётные данные по умолчанию остаются на этом устройстве. Синхронизация обычных настроек включается отдельно на каждом устройстве. Синхронизация учётных данных — единый выбор для всего аккаунта Chrome, но участвуют в ней только устройства с включённой синхронизацией настроек; остальные продолжают использовать локальные учётные данные. Для новых пользователей она по умолчанию выключена. Если при обновлении в Chrome Sync уже есть непустые учётные данные, она остаётся включённой во избежание потери данных. При включении ключи API, токены, пароли и учётные данные экспорта передаются через Chrome Sync; они только обфусцированы, но не зашифрованы. Сохранённые закладки, содержимое страниц и офлайн-очередь никогда не попадают в Chrome Sync. AI-запросы отправляются **только** через включённые или вызванные вами функции — AI-метки/аннотацию, вопросы по странице, перевод, объяснение выделенного фрагмента или необязательный обзор ключевых моментов — напрямую настроенному вами провайдеру. При установке предоставляется доступ только к Pinboard; AI, Jina, выбранные для пакетной обработки сайты и необязательные места назначения для экспорта, архивации и резервного копирования запрашивают разрешение только для конкретного сайта при выполнении соответствующего действия. Пользовательские сетевые конечные точки должны использовать HTTPS; HTTP разрешён только для `localhost`, `127.0.0.1` и `[::1]`. Страницы расширения применяют строгую Content-Security-Policy (без удалённого кода). Полная политика: <https://pine2d.github.io/Pinboard-Bookmark-Enhanced/privacy.html>
