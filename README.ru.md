# Pinboard Bookmark Enhanced

[English](README.md) | [简体中文](README.zh-CN.md) | [繁體中文](README.zh-TW.md) | [繁體中文（香港）](README.zh-HK.md) | [Deutsch](README.de.md) | [Français](README.fr.md) | [日本語](README.ja.md) | [Polski](README.pl.md) | **Русский**

Расширение Chrome, которое улучшает закладки [Pinboard](https://pinboard.in) AI-метками, аннотациями и полностью настраиваемым интерфейсом.

> **Примечание:** Требуется аккаунт [Pinboard](https://pinboard.in) — Pinboard (pinboard.in) — это независимый, **платный** сервис закладок. Это расширение является сторонним клиентом, который подключается к вашему существующему аккаунту Pinboard с помощью вашего собственного API-токена Pinboard. Оно не связано с Pinboard, не спонсируется и не одобряется им. Чтобы пользоваться этим расширением, у вас уже должен быть (или нужно оформить) платный аккаунт Pinboard.in.

## Возможности

- **Умный захват** — автозаполнение заголовка, URL, мета-описания, реферера и выделенного текста, а также удаление трекинг-параметров (`utm_*`, `gclid`, `fbclid`, …) из сохраняемого URL при захвате или вставке, с агрессивным режимом и собственными списками сохраняемых и удаляемых параметров
- **Быстрое и пакетное сохранение** — сохраняйте страницу (или в список *на потом*) прямо с горячей клавиши, не открывая попап, либо пакетно сохраняйте все открытые вкладки с AI-тегированием каждой, прогрессом в реальном времени и объединением в набор вкладок (Tab Set; ему тоже можно назначить отдельную горячую клавишу); офлайн-черновики остаются в локальной очереди, а после восстановления связи расширение автоматически повторяет сохранение в Pinboard
- **AI-метки и аннотации** — свой ключ для 13 LLM-провайдеров или любого OpenAI-совместимого эндпоинта; AI читает очищенный текст статьи — без рекламы, меню и боковых панелей
- **Инструменты для меток** — автодополнение по вашим меткам, по меткам, предложенным Pinboard, и пресеты в одно нажатие, плюс панель управления, которая находит дубликаты и редко используемые метки (эвристика + кластеризация AI по запросу) и объединяет их небольшими партиями с прогрессом в реальном времени
- **Быстрые ссылки и статус** — в одно нажатие к страницам Unread, Network, Notes и Popular; иконка на панели сразу показывает, сохранена ли текущая страница, сохранена она, или сохранена *на потом*
- **Страница в Markdown** — преобразуйте текущую страницу в чистый Markdown со встроенным предпросмотром (режимы рендеринга/исходника, оглавление, статистика чтения); копируйте или скачивайте как `.md`, стилизованный `.html` или отдельную статью `.epub` (читается на Kindle/Kobo/KOReader), настраивайте frontmatter (включая автора, дату публикации, сайт, обложку и количество слов), обработку изображений (оставить, удалить или встроить офлайн) и оглавление либо отправляйте напрямую в [Obsidian](https://obsidian.md), GitHub Gist или любой вебхук (совместимый с Readwise). Извлечение с учётом сайта сохраняет читаемость вопросов-ответов, социальных постов и форумных веток (Zhihu, Hacker News, Stack Overflow, X/Twitter, …); выберите [defuddle](https://github.com/kepano/defuddle) + [Turndown](https://github.com/mixmark-io/turndown) (локально) или [Jina Reader](https://jina.ai/reader) (облако). Изображения, заблокированные защитой сайта от хотлинкинга, распознаются в предпросмотре и восстанавливаются одним щелчком. При экспорте выводится предупреждение, если сохранённые ссылки на изображения уже не загружаются — «Встроить» упакует что сможет в скачанный файл и сообщит об остальном.
- **Вопросы и перевод страницы** — прямо в предпросмотре Markdown задавайте вопросы и получайте потоковые ответы с проверенными ссылками на источник, по которым можно перейти к нему, объясняйте или переводите любой выделенный фрагмент тут же (и сохраняйте ответ как заметку), либо переводите всю страницу с собственным глоссарием, актуальным расходом токенов и режимами оригинала / двуязычного текста / перевода
- **Инструменты для чтения** — выделяйте текст пятью цветами с заметками, которые сохраняются при повторном рендеринге, переводе и даже при изменении содержимого страницы (сместившийся текст находится заново автоматически; если найти его не удалось, заметка сохраняется и помечается — она никогда не теряется), просматривайте выделения в панели заметок (Notebook), ищите по статье (`/`, при необходимости — по регулярным выражениям; ваши заметки тоже участвуют в поиске), заглядывайте в сноски прямо в тексте, продолжайте чтение с того места, где остановились, настраивайте размер шрифта и межстрочный интервал в панели Aa, выбирайте ширину чтения (680/880/1080 px), включайте режим фокусировки, чтобы читать без отвлекающих элементов, или добавьте необязательный AI-обзор ключевых моментов (по умолчанию выключен); нажмите `?`, чтобы увидеть полный список горячих клавиш
- **Автоархивация** — при желании отправляйте каждое сохранение в [Wayback Machine](https://web.archive.org) от Internet Archive — с журналом архивации и повторными попытками, чтобы ваши ссылки оставались доступны, даже если исходная страница исчезнет
- **Темы для `pinboard.in`** — 13 тщательно подобранных палитр (Dracula · Nord · Catppuccin · Solarized · Flexoki · Gruvbox · …) плюс собственный слой CSS, который синхронизируется между устройствами, регулируемая ширина попапа и сортировка по популярности на страницах меток
- **Резервное копирование настроек** — экспортируйте и импортируйте все свои настройки (и, при желании, выделения) в виде файла или настройте автоматическое резервное копирование на собственный сервер WebDAV (например, Nextcloud), чтобы ваша конфигурация и пользовательские темы переносились между устройствами
- **9 языков** · настраиваемые горячие клавиши · локальное хранилище в приоритете · никакого трекинга

## Установка

**[→ Установить из Chrome Web Store](https://chromewebstore.google.com/detail/pinboard-bookmark-enhance/pnjndmjhljjbdlbejeenkepdalokfooh)** — рекомендуется

Или загрузите распакованное расширение из ZIP с релизом:
1. Скачайте последний [ZIP с релизом](https://github.com/pine2D/Pinboard-Bookmark-Enhanced/releases/latest)
2. Распакуйте
3. `chrome://extensions/` → включите **Режим разработчика** → **Загрузить распакованное расширение** → выберите распакованную папку

После установки: нажмите на иконку в панели инструментов → вставьте свой [API-токен Pinboard](https://pinboard.in/settings/password) → сохраните

## Конфиденциальность

Никакого трекинга, аналитики или телеметрии. Для новых пользователей настройки и учётные данные по умолчанию остаются на этом устройстве. Синхронизация обычных настроек включается отдельно на каждом устройстве. Синхронизация учётных данных — единый выбор для всего аккаунта Chrome, но участвуют в ней только устройства с включённой синхронизацией настроек; остальные продолжают использовать локальные учётные данные. Для новых пользователей она по умолчанию выключена. Если при обновлении в Chrome Sync уже есть непустые учётные данные, она остаётся включённой во избежание потери данных. При включении ключи API, токены, пароли и учётные данные экспорта передаются через Chrome Sync; они только обфусцированы, но не зашифрованы. Сохранённые закладки, содержимое страниц и офлайн-очередь никогда не попадают в Chrome Sync. AI-запросы отправляются **только** через включённые или вызванные вами функции — AI-метки/аннотацию, вопросы по странице, перевод, объяснение выделенного фрагмента или необязательный обзор ключевых моментов — напрямую настроенному вами провайдеру. При установке предоставляется доступ только к Pinboard; AI, Jina, выбранные для пакетной обработки сайты и необязательные места назначения для экспорта, архивации и резервного копирования запрашивают разрешение только для конкретного сайта при выполнении соответствующего действия. Пользовательские сетевые конечные точки должны использовать HTTPS; HTTP разрешён только для `localhost`, `127.0.0.1` и `[::1]`. Страницы расширения применяют строгую Content-Security-Policy (без удалённого кода). Полная политика: <https://pine2d.github.io/Pinboard-Bookmark-Enhanced/privacy.html>
